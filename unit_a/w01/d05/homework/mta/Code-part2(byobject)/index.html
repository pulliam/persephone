WELCOME TO MTA CALCULATOR

This Calculator is run by a function that takes 1 object.

Click Right Button and then "Inspect Element" to Start. Then Click on "Console"

You can type this into the console to calculate a route:
calculateDistance({firstLine: "L", firstStop: "Union Square", secondLine: "6", secondStop: "33rd"});
Replace the values inside quotation marks to your desired value according to:
The supported lines are: "Six", "L" or "N".
Here are the supported stops by the MTA System:
L: 8th, 6th, Union Square, 3rd, 1st
N: Times Square, 34th, 28th, 23rd, Union Square, 8th
6: Grand Central, 33rd, 28th, 23rd, Union Square, Astor Place

Happy Commute Calculating!
